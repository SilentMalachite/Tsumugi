## 条件・生成規則

`requiredWhen`はADR 0024の閉じた条件DSLだけを使用する。`always`、`optional`、`never`、および `all` / `any` / `not` / `fieldEquals` / `fieldPresent` / `fieldNonZero` / `recordPresent` / `rowPresent` / `modelPresent` / `modelTrue` / `modelNonZero` / `modelEquals` / `modelIn` / `inputPresent` / `serviceProvisionMonthBefore` / `processingMonthBefore` である。

`generatorRule`はtarget、具体的な入力field/model/selector、演算、公式sourceを持つ。自由記述による値推測は禁止する。
