# 行動規範

## 私たちの誓い

私たち（メンバー・コントリビューター・リーダー）は、年齢・体格・目に見える / 目に見えない障害・民族・性自認と性表現・経験レベル・教育・社会経済的地位・国籍・容姿・人種・カースト・肌の色・宗教・性的指向にかかわらず、誰もが嫌がらせを受けないコミュニティを実現することを誓います。

私たちはオープンで歓迎的、多様で包括的、そして健全なコミュニティに貢献するように行動し、関わり合います。

## 私たちの基準

ポジティブな環境に貢献する行動の例:

- 他者への共感と優しさを示す
- 異なる意見・観点・経験を尊重する
- 建設的なフィードバックを行い、礼儀正しく受け取る
- 自分のミスの影響を受けた人に対して責任を認め、謝罪し、その経験から学ぶ
- 個人の利益ではなくコミュニティ全体にとって最善のことを優先する

許容できない行動の例:

- 性的な言葉やイメージの使用、あらゆる種類の性的な注目や働きかけ
- 荒らし、侮辱的・軽蔑的なコメント、個人攻撃・政治攻撃
- 公的・私的なハラスメント
- 明示的な許可なしに他者の個人情報（物理または電子的な住所など）を公開すること
- 業務上不適切と合理的に考えられるその他の行動

## 適用の責任

コミュニティリーダーは、許容される行動の基準を明確化・施行する責任があり、不適切と判断する行動に対して適切で公正な是正措置を講じます。

コミュニティリーダーは、本行動規範に沿わないコメント・コミット・コード・wiki 編集・Issue 等を編集・削除する権利を有し、必要に応じて理由を説明します。

## 適用範囲

本行動規範は、コミュニティスペース内すべて、および個人がコミュニティを公的に代表している場合のすべての場所に適用されます。

## 違反の報告

虐待・嫌がらせ・その他許容できない行動を目撃または経験した場合は、メンテナへ **GitHub の Private Vulnerability Reporting** または個別連絡で報告してください。

報告は機密として扱われ、報告者の安全とプライバシーが尊重されます。

## 適用ガイドライン

コミュニティリーダーは、行動規範違反の影響を判断する際に以下のガイドラインに従います:

### 1. 訂正

**コミュニティへの影響**: 不適切な言葉遣い、その他コミュニティで非専門的または歓迎されない行為。

**結果**: コミュニティリーダーからの非公開・書面の警告。違反内容の明確化と、行動が不適切であった理由の説明。公の謝罪を求める場合があります。

### 2. 警告

**コミュニティへの影響**: 単一の事案または一連の行為による違反。

**結果**: 継続的な行動に対する結果を伴う警告。当事者との一定期間の連絡禁止を含み、本行動規範の遵守期間を求める場合があります。

### 3. 一時的禁止

**コミュニティへの影響**: 持続的な不適切行動を含む、コミュニティ基準の重大な違反。

**結果**: 一定期間のコミュニティとの一切の交流または公的コミュニケーションの一時禁止。

### 4. 永久禁止

**コミュニティへの影響**: 持続的な不適切行動・個人への嫌がらせ・特定の階級の個人に対する攻撃または侮辱を含む、コミュニティ基準違反のパターン。

**結果**: コミュニティ内のあらゆる公的交流の永久禁止。
